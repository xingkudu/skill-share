## 最小字段数要求

| 章节 | 最小要求 |
|---|---|
| 商品身份字段表 | **12+ 行** (Style/部门号/供应商/SKU×2/Kimball×2/季节/NIF/RN CODE/产地/日期×2/颜色) |
| 成分字段 | Body + Cover **各一组**,注明**14 种欧洲语言是否齐全**、**阿拉伯文 RTL 是否正确** |
| 水洗/年龄段 | **6+ 检查项** (温度/五条禁令/WASH BEFORE FIRST USE/Made in China/年龄段/特殊警告) |
| 版面结构表 | **5+ 检查项** |
| 数量表 | 按**批次/市场/尺码**分组，至少**4 行** |
| 问题清单 | 如有问题，每项必须有**根因分析** + **具体修复方案** |
| 回复建议 | **中英双语** |

## 禁止行为

- ❌ 不要只写"所有字段一致"而不列表格
- ❌ 不要省略成分字段的多语言检查
- ❌ 不要只给总数而不分组列数量明细
- ❌ 不要只说"有错误"而不分析根因和修复方案
- ❌ 不要只给中文回复而不给英文模板
